# ANN - Power Plant Energy Output Prediction (Regression)

This project was split from the monorepo notebook collection and contains one notebook with isolated dependencies.

## Files
- `artificial_neural_network - Power Plant Energy Output Prediction (Regression).ipynb`
- `requirements.txt`

## Setup
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
jupyter lab
```

This notebook uses an ANN workflow that aligns with the neural-network project style shown on
`tendaisibanda.dev` (TensorFlow/Keras model development with `scikit-learn` preprocessing).
