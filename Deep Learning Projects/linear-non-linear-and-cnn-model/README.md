# Linear / Non-linear / CNN Models

This project was split from the monorepo notebook collection and now includes both linear/non-linear/CNN notebook variants in one repository.

## Files
- `Linear,Non_linear_and_CNN_Model.ipynb`
- `Linear,_Non_linear_and_CNN_Model.ipynb`
- `requirements.txt`

## Setup
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
jupyter lab
```

## Portfolio Context
This combined repo mirrors the model-comparison style presented in projects on
`tendaisibanda.dev`, where linear/non-linear/CNN approaches are evaluated with
clear performance tradeoffs.
