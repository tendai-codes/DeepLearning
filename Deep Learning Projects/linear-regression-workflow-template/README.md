# Linear_Regression_Workflow_Template

This project was split from the monorepo notebook collection and contains one notebook with isolated dependencies.

## Files
- `Linear_Regression_Workflow_Template.ipynb`
- `requirements.txt`

## Setup
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
jupyter lab
```

## Portfolio Context
This workflow template aligns with the end-to-end experimentation style shown on
`tendaisibanda.dev`: data setup, model training, evaluation, and iteration in a
reproducible PyTorch notebook flow.
