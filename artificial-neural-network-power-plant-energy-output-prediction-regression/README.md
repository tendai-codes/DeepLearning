# Power Plant Energy Output Prediction Using Artificial Neural Networks

## Problem

This project analysed operational power plant sensor data to model relationships between environmental conditions and energy output in order to support prediction of plant electricity generation.

## Objective

- Prepare environmental and operational variables for regression modelling
- Train an artificial neural network to estimate energy output
- Examine relationships between input features and predicted generation values
- Compare predicted energy output with observed measurements

## Approach

- Imported and structured power plant operational dataset for supervised learning
- Split predictor variables and target energy output values into training and testing sets
- Trained an artificial neural network regression model on historical observations
- Generated predictions and compared estimated output with actual energy generation values

## Key Findings

- The artificial neural network produced energy output estimates from environmental input variables
- Model predictions followed observable patterns in recorded plant output values across the evaluation dataset
- Feature scaling supported stable neural network training behaviour
- Visual comparisons enabled inspection of differences between predicted and observed energy output values
