# Image Classification Using Convolutional Neural Networks

## Problem

This project analysed labelled image data to develop a convolutional neural network capable of classifying visual categories based on extracted spatial features.

## Objective

- Prepare image datasets for convolutional neural network training
- Construct a CNN architecture for supervised image classification
- Train the model on labelled image samples
- Evaluate predicted classes against observed labels

## Approach

- Loaded and structured image datasets for training and evaluation
- Applied image preprocessing steps suitable for CNN input
- Built and trained a convolutional neural network model
- Generated predictions and evaluated classification outputs on test images

## Key Findings

- The convolutional neural network produced class predictions from image feature representations
- Convolutional layers extracted spatial patterns relevant to classification tasks
- Model evaluation outputs enabled comparison between predicted and actual labels
- Image preprocessing supported compatibility with CNN training workflows
