# Linear, Non-Linear, and Convolutional Neural Network Model Comparison

## Problem

This project analysed image classification data to compare the behaviour of linear, non-linear, and convolutional neural network models for predicting visual categories from image inputs.

## Objective

- Train a linear model for baseline image classification performance
- Implement a non-linear neural network to capture higher-order feature relationships
- Apply a convolutional neural network to extract spatial image features
- Compare prediction behaviour across model architectures

## Approach

- Prepared image tensor data for supervised classification modelling
- Trained a linear classifier using flattened image inputs
- Implemented a non-linear neural network with hidden-layer transformations
- Built and evaluated a convolutional neural network using spatial feature extraction layers

## Key Findings

- Linear models produced baseline classification predictions from flattened pixel representations
- Non-linear neural networks captured additional feature relationships beyond linear transformations
- Convolutional neural networks extracted spatial image patterns relevant to classification tasks
- Model outputs enabled comparison of prediction behaviour across linear, non-linear, and CNN architectures
