# Natural Language Processing Text Analysis

## Problem

This project analysed textual data to extract structured representations from raw language input in order to support downstream natural language processing tasks.

## Objective

- Prepare raw text data for computational analysis
- Apply preprocessing techniques to clean and standardise text
- Convert text into numerical feature representations
- Support interpretation of text structure using transformed features

## Approach

- Loaded and inspected raw textual dataset
- Applied text cleaning steps including tokenisation and normalisation
- Removed stopwords and non-informative tokens
- Converted processed text into vectorised representations for analysis

## Key Findings

- Text preprocessing transformed raw language inputs into structured token sequences
- Stopword removal reduced non-informative vocabulary from the dataset
- Tokenisation enabled consistent segmentation of text for analysis workflows
- Vectorisation produced numerical representations suitable for downstream NLP modelling tasks
